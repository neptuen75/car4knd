📋 Google Sheets 연동 설정 방법

  1. Google Sheets에서 새 스프레드시트 생성
  2. 상단 메뉴 확장 프로그램 → Apps Script 클릭 (스프레드시트 안에서!)
  3. 에디터 기존 코드 전체 삭제 후 Code.gs 내용 붙여넣기 → 저장(💾)
  4. 배포 → 새 배포 / 유형: 웹 앱 / 실행: 나 / 액세스: 모든 사용자
  5. 배포 → 권한 허용 → 웹앱 URL 복사 (/exec 로 끝나는 것)
  6. 이 페이지 상단 입력창에 URL 붙여넣기 → 연결하기
💡 코드 수정 후 재배포: 배포 관리 → ✏️ 편집 → 버전: 새 버전 으로 업데이트해야 URL이 유지됩니다.
⚠️ 저장 실패 시: URL이 /exec 로 끝나는지, 액세스 권한이 "모든 사용자" 인지 확인하세요.
